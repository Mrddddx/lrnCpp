##### 函数的不同
> 函数原型c和c++要求不一样
> c++ 的 main 必须是int ,一般 return 0 代表正常 -1 代表不正常
##### inline
> 相当于把函数的语句插在调用位置那里
> 一般不能有for 和 switch 等
> 一般比较短小
```cpp
#define dou(x) x * 2 
dou(1 + 2)
//结果是5!
```
> 默认参数
```cpp

```
